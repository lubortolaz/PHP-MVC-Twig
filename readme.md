# PHP Twig MVC Base Project

This is a simple PHP MVC project template with Twig and Sass.

````
.
├── config/
│   └── config.php
├── Controller/
│   ├── AbstractController.php
│   └── DefaultController.php
├── Core/
│   ├── Database/
│   ├── Helper/
│   ├── Model/
│   └── Service/
├── logs/
├── public/
│   ├── css/
│   ├── fonts/
│   ├── images/
│   ├── js/
│   └── index.php
├── sass/
└── View/
    ├── default/
    │   └── index.html.twig
    └── base.html.twig
````
Requirement : npm, composer

To initialize : `make init`
